# hotel-management-api-backend

A backend API built using:
- Node.js
- Express.js
- MongoDB
- Mongoose

Features
- Create Person and Menu
- Get All Persons and Menu Items
- Update Person and Menu Items
- Delete Person and Menu Items

## API Endpoints
### GET
- /person
- /menu

### POST
- /person
- /menu

### PUT
- /person/:id
- /menu/:id

### DELETE
- /person/:id
- /menu/:id
